Self-Practice Platform

Catch, patch and test for security bugs anytime, anywhere.

Story Telling

We love stories. We learn from stories. Stories stick in our head

We don't bombard you with confusing security terms. We tell you stories of real-world incidents that was caused by security bugs. Through the story you will learn the internals of the bug, how to detect and eliminate it.

Clone, Add, Commit, Push

We made it so that you don't need to learn a new tool just to start learning secure programming.

You use the same familiar tools that you use for development to learn our courses.

Write Security Tests

A security bug is a run-time exception that we have missed to test and handle

You will learn how to write security tests for classes of security bugs. Your goal in each lab is simple, fix the program so it passes the tests.
